В папке "Selenium" лежат скрипты одноименного фреймворка:
Selenium_first - первый опыт использования Selenium, ищет скидки на гиковские фигурки на одном сайте;
Maps_parser - второй опыт с парсингом. Принимает n-ое кол-во адресов, актуальных для Екатеринбурга, затем создает их всевозможные комбинации и находит длину маршрутов по Яндекс.Картам. Результат сохраняет в Excel-таблицу в папку со скриптом. Планирую доделать так, чтобы высчитывал наименее затратный по времени/расстоянию маршрут.

В папке "VGIK" лежит часть исходного кода сайта, который я писал на Node.js с Express, MongoDB и Handlebars, но вскоре его решили перевести на WordPress.
В итоге, в папке "VGIK_gulp" лежит его фронтенд.
